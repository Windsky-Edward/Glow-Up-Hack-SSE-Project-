# Glow Up Hack

AI-Powered Beauty Intelligence Marketplace.

This is a single-file static landing page for the Glow Up Hack university innovation project. It can be deployed directly on Vercel as a static HTML site.

## Deploy on Vercel

1. Upload `index.html` and `README.md` to the GitHub repository root.
2. Open Vercel and import the GitHub repository.
3. Keep the framework preset as `Other` or static HTML.
4. Deploy without a build command.
